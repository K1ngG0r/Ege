# Explorer



Нуууу легкая версия проводника.





Тут реализованы:
      
      1) Показ содержимое дисков
      
      2) Создание папок/файлов
      
      3) Удаление папок/файлов
      
      4) Переиминовывание папок/файлов
      
      5) Копирование/перенос папок/файлов
      
      6) Вычисление размера папок/файлов
      
      7) Поиск по маске
